# Project Intelligence Hub - iPhone Free App

## Status

iPhone Free App: Ready as PWA / Add to Home Screen.

Native iOS IPA: Not generated.

## Stable URL

https://samco-mob-intelligence-dashboard.streamlit.app

## iPhone Installation Steps

1. Open Safari on iPhone.
2. Open this link:

   https://samco-mob-intelligence-dashboard.streamlit.app

3. Tap Share.
4. Tap Add to Home Screen.
5. Name it Project Intelligence Hub.
6. Tap Add.
7. Open Project Intelligence Hub from the iPhone Home Screen.
8. Login using the provided username and password.

## Important

This is the free iPhone app route.

A native iOS IPA requires macOS, Xcode, and Apple signing.
