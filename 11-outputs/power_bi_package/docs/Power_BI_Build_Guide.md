# Project Intelligence Hub - Power BI Build Guide

This ZIP is a Power BI build kit for recreating the Project Intelligence Hub executive reporting layer in Power BI.

## Included
- data/projects_registry.csv: discovered projects and folder paths.
- data/source_file_register.csv: all project source files for lineage.
- data/claims_intelligence_summary.csv: per-project Claims Intelligence counts.
- data/contract_clauses.csv: extracted clause library by project.
- dax/measures.dax: core executive and claims measures.
- power_query/project_intelligence_hub_queries.pq: Power Query starter script.
- theme/project_intelligence_hub_theme.json: navy / teal / gold executive theme.
- layout/report_pages.md: page-by-page visual specification.

## Build Steps
1. Open Power BI Desktop.
2. Import the theme JSON from View > Themes > Browse for themes.
3. Load the CSV files from the data folder.
4. Create relationships:
   - Projects[project_id] 1:* Source Files[project_id]
   - Projects[project_id] 1:* Claims Summary[project_id]
   - Projects[project_id] 1:* Contract Clauses[project_id]
5. Add the measures from dax/measures.dax.
6. Build pages according to layout/report_pages.md.
7. Publish to Power BI Service if required.

## Refresh Method
Regenerate this package from the Streamlit app or refresh the CSV exports, then refresh Power BI.
For live local alignment, point Power Query directly to the CSV folder generated by the app.

## Important Limitation
Power BI cannot run Streamlit Python UI interactions directly. This package recreates the reporting, dashboards, data model, claims KPIs, lineage, and executive visuals in Power BI using the same project data outputs.
